10.react_route

约 289 字小于 1 分钟

2025-08-28

1.Router的基本使用

  1. 安装react-router-dom，npm install react-router-dom
  2. 使用BrowserRouter使用history模式，或者HashRouter使用hash模式。
<HashRouter>
  <App />
</HashRouter>
  1. 路由映射配置。 Routes：包裹所有的Route，在其中匹配一个路由 Route：Route用于路径的匹配。里面path属性用于设置匹配到的路径，element用于设置匹配路径后渲染的组件。
<Routes>
  <Route path='/home' element={<Home />}></Route>
  <Route path='/about' element={<About />}></Route>
</Routes>
  1. 路由配置和跳转
<Link to="/home">首页</Link>
<Link to="/about">关于</Link>

Navigate导航

用于路由的重定向。

<Route path='/' element={<Navigate to="/home"/>}  />

Not Found页面配置

<Route path='*' element={<NotFound />}  />

路由的嵌套

这里的重点是要在父路由元素中使用====作为子路由的占位元素。

<Routes>
  <Route path='/home' element={<Home />}>
    <Route path='/home/abc' element={<Abc />}  />
  </Route>
</Routes>

<h1>Home</h1>
<Link to="/home/abc">首页</Link>
<Outlet/>

参数传递

  1. 动态路由的方式，比如 /detail/：id。
获取：
const params = useParams()
  1. search传递参数，
获取：
const [searchParams] = useSearchParams()
const query = Object.fromEntries(searchParams)
